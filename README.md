# DNA Sequence Analyzer 🧬

A Python project that analyzes multiple DNA sequences.

## Features

- Analyze multiple DNA samples
- Count A, T, G, and C nucleotides
- Detect invalid nucleotides
- Object-oriented design
- Modular code structure
- Exception handling
- Allows users to enter an additional DNA sequence

## Technologies Used

- Python
- Lists
- Loops
- Functions
- Classes
- Modules
- Dictionaries
- Exception Handling
